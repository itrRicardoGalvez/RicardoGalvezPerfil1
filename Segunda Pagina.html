Componentes Activos:

Listado de componentes

Switch

Dispositivo que conecta múltiples dispositivos dentro de una red, gestionando el tráfico de datos.

Funcionalidad: Opera en diferentes capas del modelo OSI, segmentando y gestionando el tráfico de red.

Router

Dispositivo que conecta redes diferentes y dirige el tráfico de datos entre ellas. También realiza funciones de seguridad como firewall.

Aplicaciones: Esencial para conectar redes domésticas o de oficina a Internet.

Tarjeta de Red (NIC)

Permite a un dispositivo conectarse a una red física o inalámbrica, actuando como interfaz entre el dispositivo y la red.

Interruptor Inteligente Wifi

Dispositivo domótico que permite el control remoto de dispositivos eléctricos a través de una red inalámbrica.

Integración con sistemas de automatización y control por aplicaciones móviles o asistentes de voz.

Access Point

Permite a dispositivos inalámbricos conectarse a una red cableada, funcionando como puerta de entrada entre redes.

Soporta múltiples estándares de Wi-Fi y ofrece funciones avanzadas como la gestión de usuarios y seguridad.

Herramientas utilizadas para realizar cableado estructurado:

Listado de herramientas

Ponchadora

Herramienta para insertar y asegurar cables en jacks o paneles.

Es esencial para la instalación de cables en conectores, asegurando una conexión firme y fiable.

Crimpadora

Herramienta para fijar conectores RJ45 en los extremos de un cable UTP.

Es esencial para crear cables de red personalizados con las configuraciones necesarias.

Desforradora

Herramienta para quitar el recubrimiento de cables sin dañar los hilos internos.

Fundamental para preparar los cables antes de la crimpación o conexión en paneles.

Tester/Probador de Cables

Dispositivo para verificar la correcta conexión y funcionalidad de los cables de red.

Crucial para asegurar que las conexiones se han realizado correctamente, sin cortocircuitos o cruces de cables.

¿Qué es el cableado horizontal?

El cableado horizontal es la infraestructura que conecta las estaciones de trabajo (como computadoras, teléfonos, y otros dispositivos de red) con el cuarto de telecomunicaciones dentro de la misma planta de un edificio.

Es utilizado para conectar dispositivos como computadoras, impresoras, teléfonos VoIP, y otros equipos de red a la infraestructura principal de telecomunicaciones.
